# Container Engine

## O que é um container engine?

Um container engine é um software que permite criar, executar e gerenciar containers. Ele fornece uma interface para interagir com os containers, permitindo que os usuários criem, iniciem, parem e excluam containers de forma fácil e eficiente. O container engine é responsável por gerenciar os recursos do sistema, como CPU, memória e armazenamento, para garantir que os containers sejam executados de forma eficiente e segura. O Docker é um exemplo popular de container engine, mas existem outros, como o Podman e o CRI-O.
