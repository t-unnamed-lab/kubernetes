# Open Container Initiative (OCI)

## O que é OCI?

OCI (Open Container Initiative) é um projeto aberto que define padrões para containers e runtimes de containers. O objetivo do OCI é garantir a interoperabilidade entre diferentes ferramentas e plataformas de containers, promovendo um ecossistema de containers mais consistente e confiável. O OCI define especificações para imagens de containers, runtimes de containers e outros componentes relacionados, permitindo que os desenvolvedores e operadores usem diferentes ferramentas de forma compatível.
