# Conhecendo o kubectl attach e exec

## kubectl attach

O comando `kubectl attach` é usado para se conectar a um container em execução em um pod e interagir com ele. Ele permite que você veja a saída do container e envie comandos para ele. O comando é útil para depurar problemas com um container ou para executar comandos interativos dentro do container.

## kubectl exec

O comando `kubectl exec` é usado para executar um comando em um container em execução em um pod. Ele permite que você execute comandos dentro do container e veja a saída. O comando é útil para depurar problemas com um container ou para executar tarefas administrativas dentro do container.
