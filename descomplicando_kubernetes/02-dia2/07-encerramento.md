# Encerramento dia 2

Hoje vimos o que são os Pods, como eles funcionam e como criar um Pod com múltiplos containers. Vimos também o que são os recursos de CPU e memória, e como configurar os limites e requests de recursos para os containers. Além disso, aprendemos sobre volumes, especificamente o volume do tipo `EmptyDir`, e como ele pode ser usado para armazenar dados temporários dentro de um Pod.
