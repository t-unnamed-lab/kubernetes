# os sensacionais kubectl get pods e kubectl describe pods

## kubectl get pods

O comando `kubectl get pods` é usado para listar os pods em um cluster Kubernetes. Ele exibe informações básicas sobre os pods, como o nome, o status, a idade e o número de containers em execução. O comando pode ser personalizado usando opções como `-o wide` para exibir informações adicionais, como o nó em que o pod está sendo executado e os endereços IP dos containers.

## kubectl describe pods

O comando `kubectl describe pods` é usado para exibir informações detalhadas sobre um pod específico. Ele fornece uma visão abrangente do pod, incluindo informações sobre os containers, volumes, eventos e status do pod. O comando é útil para diagnosticar problemas com um pod, como falhas de inicialização ou problemas de rede. Ele também pode ser usado para obter informações sobre os recursos alocados para o pod e os limites de recursos.
